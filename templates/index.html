🤖 SmartBackup Setup

Follow these simple steps to get your backup system running

1

Create Your Telegram Bot

•

Open Telegram and search for @BotFather

•

Send the command /newbot

•

Follow the prompts to name your bot (e.g., "My Backup Bot")

•

Copy the Bot Token when BotFather provides it (it looks like: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz)

2

Get Your Chat ID

•

Start a chat with your new bot (send any message)

•

Visit this URL in your browser (replace YOUR_TOKEN with your actual token):

https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
•

Look for "chat":{"id":123456789 in the response - that number is your Chat ID

3
